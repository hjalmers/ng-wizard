Use the status property of a step to highlight and clarify wizard progress - completed steps, errors etc.

By default steps will be marked as finished or complete when you've passed them. If you want to handle step states manually it's possible to do so using the markPassedAsSuccess config option. It's also possible to set other states using the setState method.
